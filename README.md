# DB-Project
This is a python program that stores and maintains a file to ensure that regardless if the program is stopped it will always retain the data that was input

# Setup
If ran in visual studio simple run the code through VS code window.
If ran through terminal run command python3 "Database.py"

The program will not display text but will accept inputs from the user.
User should use 
"SET <key> <value>" 
to store information in the database file.

User should use 
"GET <key>" 
to retrieve possible keys stored within the program.

When done the user should use 
"EXIT" 
to terminate the program.
